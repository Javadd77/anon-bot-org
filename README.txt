
📦 راهنمای نصب ربات در Railway

1. وارد سایت شوید:
   👉 https://railway.app/

2. وارد حساب کاربری‌تان شوید یا ثبت‌نام کنید.

3. روی "New Project" کلیک کنید → "Deploy from GitHub" یا "Deploy from template".

4. اگر GitHub نداری، این فایل‌ها را دستی آپلود کن:
   - anon_bot.py
   - requirements.txt

5. بعد از آپلود:
   - وارد بخش Settings شو.
   - از بخش "Variables"، یک متغیر به اسم TOKEN بساز و مقدار آن را توکن ربات قرار بده.

6. در بخش "Deployments"، پروژه را اجرا کن. بعد از چند ثانیه ربات بالا می‌آید.

🔁 هر بار تغییرات دادی، Deploy مجدد کن یا Auto Deploy رو فعال کن.
